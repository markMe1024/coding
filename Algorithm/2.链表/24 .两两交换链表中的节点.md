我觉得这个问题比较简单，没有太多的技巧在里面，处理好节点交换逻辑就可以

代码：

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        ListNode dummyNode = new ListNode(0, head);

        ListNode p = dummyNode;
        ListNode n = null;
        ListNode tmp = null;

        while (p.next != null && p.next.next != null) {
            n = p.next.next.next;

            tmp = p.next;
            p.next = p.next.next;
            p.next.next = tmp;
            tmp.next = n;

            p = p.next.next;
        }

        return dummyNode.next;
    }
}
```
